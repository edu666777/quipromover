➕ 1. Reação de Síntese ou Adição

Juntando as Peças

Ocorre quando duas ou mais substâncias se unem para formar apenas uma substância nova e mais complexa.

A + B ➔ AB

Exemplo: C + O₂ ➔ CO₂

✂️ 2. Reação de Análise ou Decomposição

Separando as Partes

É o exato oposto da Síntese. Uma única substância divide-se em duas ou mais substâncias mais simples.

AB ➔ A + B
Nomes Especiais:
🔥 Pirólise: Decomposição pelo calor (símbolo Δ).
⚡ Eletrólise: Decomposição pela eletricidade.
☀️ Fotólise: Decomposição pela luz (símbolo λ).
🔄 3. Reação de Simples Troca

O Deslocamento

Ocorre quando uma substância simples reage com uma composta, "expulsando" um dos elementos e assumindo o seu lugar.

A + BC ➔ AC + B

Exemplo: Zn + 2HCl ➔ ZnCl₂ + H₂
Nota: Só acontece se o elemento "A" for mais reativo que o elemento "B".

🔀 4. Reação de Dupla Troca

A Troca de Casais

Duas substâncias compostas trocam os seus íons entre si, formando duas novas substâncias compostas.

AB + CD ➔ AD + CB

Exemplo: NaCl + AgNO₃ ➔ AgCl + NaNO₃
(A clássica reação de neutralização Ácido + Base ➔ Sal + Água também é uma dupla troca!)

Condições para ocorrer: Uma dupla troca só acontece se formar pelo menos um produto que seja:
1. Um precipitado (sólido insolúvel que vai para o fundo).
2. Um gás (substância volátil).
3. Uma substância mais fraca (menos ionizada), como a água.
